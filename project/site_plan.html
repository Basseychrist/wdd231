SITE NAME

Bassey Geo-Explorer

Site Purpose

The purpose of Bassey Geo-explorer is to provide users with an engaging way to explore the world’s geography. The site allows users to access geographic data (such as population, area, and languages), real-time weather information, and geographical facts through interactive maps and external APIs. It also offers filtering options and allows users to store preferences locally.

Scenarios

1️⃣ How can I check the capital, population, and languages of countries in Africa?

◾ This question targets users curious about learning geography by continent or country.

2️⃣ What is the current weather like in Tokyo, Japan?

◾ This addresses users looking for real-time weather information via the interactive map.

3️⃣ Can I save my temperature settings to always show in Fahrenheit instead of Celsius?

◾ This caters to users who want to customize their experience using localStorage.
